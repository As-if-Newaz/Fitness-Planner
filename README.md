# GoZayaan-Submission

A Frontend aI powered Fitness Planner project assigned by the GoZayaan tech recruitment team
Description:
A simple web application where a user provides basic personal and fitness information,
sends it to an AI model, and the response is displayed as a visual weekly fitness plan using cards and UI elements.

### Technical Spec:

- Web framework: Vanilla JS.
- No UI component library or CSS library like Tailwind. No Shadcn, MUI or Ant Design has been used.
- The system prompt that instructs the model to return structured JSON was written.
- Target design has been followed for UI designing.
- If the API fails or returns malformed JSON, the app does not break.
- The application is responsive and usable on mobile.

AI Model:
For the AI model, one of OpenRouter's free tier model `openai/gpt-oss-120b:free` has been used.
